# XM Cloud Starter Kit (Next JS)

## QUICK START

1. In an ADMIN terminal:

    ```ps1
    .\init.ps1 -InitEnv
    ```

2. Restart your terminal and run:

    ```ps1
    .\up.ps1
    ```

## About this Solution

This solution is designed to help developers learn and get started quickly
with XMCLoud + SXA.
